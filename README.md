#  Observação

O projeto consiste em um sistema de **comércio eletrônico para venda de
livros**, contendo funcionalidades como:

-   cadastro e listagem de produtos
-   carrinho de compras
-   cálculo de frete
-   processamento de pedidos
-   envio de notificações por email
